# Spike conformational dynamics

Spike conformational dynamics analyses were performed using the dedicated repository:

https://github.com/lcasalino/spike-conformational-dynamics

This repository contains the VMD/Tcl scripts used to quantify:

- hip tilting
- knee tilting
- ankle tilting
- NTD triangle area
- RBD–central-helix distance

The analyses were performed for both the RAV aerosolized virion simulations and the single-spike reference simulations.

The spike-dynamics repository also contains the plotting notebook and processed time-series outputs used for the manuscript figure generation.
